# HP Smart Tank 750 ADF 自动扫描程序

## 快速开始

### 1. 启动守护程序

```bash
cd ~
./scanner_control.sh start
```

### 2. 使用方法

1. **将纸张放入扫描仪的 ADF（自动输稿器）**
   - 打开顶部输稿器盖子
   - 纸张面朝上，顶部先进
   - 最多可放 35 张

2. **程序自动检测并扫描**
   - 检测到纸张后立即开始扫描
   - 所有页面自动连续扫描
   - 每张纸作为一个单独文件保存

3. **查看扫描结果**
   - 文件保存在 `~/测试图片/<开始时间>/` 目录下
   - 文件名格式：`page_001.jpg`, `page_002.jpg` ...
   - 扫描完成后自动打开文件夹

### 3. 控制命令

```bash
# 查看状态
./scanner_control.sh status

# 查看实时日志
./scanner_control.sh logs

# 停止守护程序
./scanner_control.sh stop

# 重启
./scanner_control.sh restart
```

## 功能特点

- ✅ **自动检测**：实时监测 ADF 纸张状态
- ✅ **连续扫描**：一次放入多张纸，自动连续处理
- ✅ **时间命名**：每次扫描创建独立文件夹，以开始时间命名
- ✅ **高清晰度**：300dpi 彩色扫描
- ✅ **格式支持**：JPEG 格式（兼容性好，文件适中）
- ✅ **后台运行**：启动后可关闭终端，持续运行

## 故障排除

### 程序无法启动
```bash
# 检查依赖
./scanner_control.sh install

# 手动测试连接
python3 hp_scanner_daemon.py
```

### 扫描仪未响应
- 确认扫描仪 IP 为 `192.168.3.11`
- 检查扫描仪是否开机并联网
- 查看日志：`./scanner_control.sh logs`

### 扫描失败
- 确保纸张放置正确（面朝上，不歪斜）
- 检查是否一次放入过多纸张（最多35张）
- 纸张不要太厚或太薄

## 退出程序

按 `Ctrl+C` 或在另一个终端运行：
```bash
./scanner_control.sh stop
```

## 技术细节

- **协议**：eSCL (AirPrint Scan)
- **分辨率**：300dpi
- **色彩**：RGB24 真彩色
- **接口**：HTTP API (192.168.3.11)
- **依赖**：Python 3, requests, Pillow
